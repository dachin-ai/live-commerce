# TikTok数据导入测试数据模板说明

## 📋 Excel文件格式要求

### 必需字段（至少包含以下之一）
- **日期**：直播日期
- **总观看人数** 或 **GMV** 或 **总订单数**：至少有一个数值字段
- **直播时长**：时长（分钟）

### 推荐字段（用于完整分析）
- 日期
- 直播ID
- 直播标题
- 总观看人数
- GMV（总成交额）
- 总订单数
- 成交订单数
- 直播时长（分钟）
- 总互动数
- 转化率

## 📝 创建测试Excel文件的步骤

### 方法1：使用Excel
1. 打开Excel
2. 在第一行输入表头（参考上面的模板）
3. 从第二行开始输入数据
4. 保存为.xlsx格式

### 方法2：使用在线工具
1. 使用Google Sheets或Office Online
2. 创建表格
3. 下载为Excel格式

---

## ⚠️ 注意事项

1. **第一行必须是表头**：系统使用第一行识别字段
2. **数据从第二行开始**：第一行之后的所有行都会被解析
3. **数值格式**：
   - 数字字段应该是数字格式，不是文本
   - 可以包含千分位逗号（系统会自动处理）
   - 可以包含货币符号（系统会自动移除）
4. **时长单位**：系统假设时长为分钟，会自动转换为小时
5. **百分比**：转化率等百分比字段可以包含%符号
