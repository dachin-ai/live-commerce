# 安装依赖说明

## 📦 新增依赖

为了支持TikTok数据导入功能，需要安装以下依赖：

### 后端依赖

```bash
cd backend
npm install xlsx
```

或者手动添加到 `backend/package.json`：

```json
{
  "dependencies": {
    "xlsx": "^0.18.5"
  },
  "devDependencies": {
    "@types/xlsx": "^0.0.36"
  }
}
```

---

## ✅ 安装步骤

1. **打开终端**，进入项目根目录

2. **安装后端依赖**：
   ```bash
   cd backend
   npm install
   ```

3. **验证安装**：
   ```bash
   npm list xlsx
   ```
   应该显示 xlsx 的版本号

4. **更新数据库结构**：
   ```bash
   npm run db:update-seed
   ```

5. **重启后端服务**：
   - 如果使用 `快速启动.bat`，关闭后重新运行
   - 如果手动启动，重启后端服务

---

## 🔍 验证安装

安装完成后，启动后端服务，检查控制台是否有错误。

如果看到以下错误：
```
⚠️  xlsx库未安装，请运行: npm install xlsx
```

说明依赖未正确安装，请重新执行安装步骤。

---

## 📝 注意事项

- 如果使用npm镜像源，可能需要配置：
  ```bash
  npm config set registry https://registry.npmmirror.com
  ```

- 如果安装失败，可以尝试：
  ```bash
  npm install xlsx --legacy-peer-deps
  ```
