## 跨项目检查点

> [!WARNING]
> 如果本次改动涉及以下内容，上下游文档**都要对齐**：

- **Coze 智能体 ↔ 后端 API**：修改了后端给 Coze 的入参 → 检查 `docs/Coze对接说明-*.md` + Coze 端配置
- **前端 ↔ 后端 API**：新增/修改接口 → 检查前端调用代码 + API 文档
- **数据库 ↔ 数据导入**：改了表结构 → 检查 Excel 解析逻辑 + 导入指南
