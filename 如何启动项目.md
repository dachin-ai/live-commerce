# 如何启动项目

## 🚀 方法1：PowerShell 脚本（推荐）

### 方式A：右键运行
1. 找到文件：`启动.ps1`
2. **右键点击** `启动.ps1`
3. 选择 **"使用 PowerShell 运行"** 或 **"Run with PowerShell"**

### 方式B：PowerShell 命令行
1. 按 `Win + X`，选择 **"Windows PowerShell"** 或 **"终端"**
2. 输入以下命令：
   ```powershell
   cd "d:\Work space\lvbcsym"
   .\启动.ps1
   ```

### 方式C：从文件资源管理器
1. 打开文件资源管理器，进入项目目录
2. 在地址栏输入：`powershell`
3. 按回车，会打开 PowerShell 窗口
4. 输入：`.\启动.ps1`

---

## 🚀 方法2：批处理脚本（备用）

### 如果 PowerShell 无法运行
1. 双击 `start.bat`
2. 如果显示问号或乱码，使用方法1

---

## 🚀 方法3：直接命令行（最简单）

### 启动前后端
打开命令提示符（cmd）或 PowerShell，执行：
```cmd
cd "d:\Work space\lvbcsym"
npm run dev
```

### 仅启动后端
```cmd
cd "d:\Work space\lvbcsym\backend"
npm run dev
```

### 仅启动前端（新窗口）
```cmd
cd "d:\Work space\lvbcsym\frontend"
npm run dev
```

---

## ⚠️ 如果 PowerShell 提示"无法执行脚本"

如果看到错误："无法加载文件，因为在此系统上禁止运行脚本"

### 解决方法：
1. 以**管理员身份**打开 PowerShell
2. 执行：
   ```powershell
   Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
   ```
3. 输入 `Y` 确认
4. 然后运行 `.\启动.ps1`

## 📋 菜单说明

启动后会显示菜单：
- `[1]` - 启动前后端
- `[2]` - 仅启动后端
- `[3]` - 检查状态
- `[4]` - 修复依赖
- `[5]` - 数据库管理
- `[6]` - 打开测试页面
- `[7]` - 打开API测试终端
- `[0]` - 退出

---

## 💡 推荐流程

1. **日常启动**（最简单）：
   - **双击 `快速启动.bat`** → 自动启动前后端

2. **首次使用或需要其他功能**：
   - 运行 `启动.ps1` → `[4] Fix Dependencies` → 等待安装完成
   - 然后 `[1] Start All` → 启动服务

3. **如果启动脚本有问题**：
   - 直接使用命令行：`npm run dev`
