## 🧪 测试步骤

### 测试1: 店铺创建和品牌策略生成

1. 访问 http://localhost:5173
2. 点击"创建店铺"按钮
3. 填写店铺信息：
   - 店铺名称：测试店铺
   - 平台：选择"抖音"
   - 选择商品分类（一级、二级、三级）
   - 选择国家/地区
   - 设置价格区间
   - 设置目标人群
   - 选择品牌定位
4. 点击"AI自动生成品牌策略"按钮
5. **验证**：生成的策略应包含：
   - 平台特色分析
   - 价格定位建议
   - 目标人群策略
   - 品牌定位建议
   - 商品分类建议
   - 区域特色建议
   - 综合建议

### 测试2: 数据筛选功能

1. 在Dashboard页面选择店铺
2. 点击"筛选数据"按钮
3. 选择不同的数据项目（如：只选择GMV和订单数）
4. **验证**：统计卡片应只显示选中的数据项目
5. 选择不同的时间周期
6. **验证**：时间周期标签正确更新

### 测试3: 布局设置功能

1. 点击右下角的设置按钮（齿轮图标）
2. 测试功能区显示/隐藏：
   - 关闭"店铺列表"
   - **验证**：店铺列表隐藏
   - 关闭"数据统计"
   - **验证**：数据统计区域隐藏
   - 关闭"任务管理"
   - **验证**：任务管理隐藏
   - 关闭"AI功能"
   - **验证**：AI功能区域隐藏
   - 关闭"图表"
   - **验证**：图表隐藏
3. 测试功能区大小调整：
   - 调整店铺列表列数（1-6）
   - 调整数据统计列数（1-9）
   - 调整任务管理列数（1-6）
   - **验证**：布局正确更新
4. 测试图标显示/隐藏：
   - 关闭"显示图标"
   - **验证**：所有图标隐藏
   - 打开"显示图标"
   - **验证**：所有图标显示
5. 点击"重置为默认"
   - **验证**：所有设置恢复默认值
6. 刷新页面
   - **验证**：设置保持（localStorage保存）

### 测试4: 智能生成任务

1. 在Dashboard页面选择店铺
2. 在任务列表点击"智能生成刷新"按钮
3. **验证**：生成的任务应包含：
   - 具体的数据指标（当前值、目标值）
   - 预计效果（量化）
   - 具体措施（3条以上）
   - 避免空泛表达

### 测试5: 角色权限

#### 测试主播角色
1. 使用主播账号登录
2. **验证**：
   - Dashboard不显示任务管理面板
   - Dashboard显示专属提示卡片
   - 分析页面不显示店铺对比
   - 执行工具页面显示受限功能

#### 测试运营角色
1. 使用运营账号登录
2. **验证**：
   - Dashboard显示完整功能
   - 分析页面显示店铺对比
   - 执行工具页面显示所有功能

#### 测试管理角色
1. 使用管理员账号登录
2. **验证**：
   - 所有功能可访问
   - 可访问用户管理页面

### 测试6: 导航和路由

1. 测试导航菜单：
   - 点击"店铺管理"
   - **验证**：跳转到 `/`
   - 点击"数据分析"
   - **验证**：跳转到 `/analysis`
   - 点击"执行工具"
   - **验证**：跳转到 `/tools`
2. 测试路由保护：
   - 退出登录
   - 访问 `/`
   - **验证**：重定向到 `/login`
   - 登录后访问 `/admin`
   - **验证**：非管理员重定向到 `/`

### 测试7: 错误处理

1. 测试API错误：
   - 停止后端服务
   - 点击任意AI功能按钮
   - **验证**：显示友好的错误提示
2. 测试网络错误：
   - 断开网络
   - 点击按钮
   - **验证**：显示网络错误提示
3. 测试401错误：
   - 清除token
   - 访问页面
   - **验证**：自动跳转到登录页
