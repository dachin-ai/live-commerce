## 三、使用说明

- **给 LLM 的文档**：核对是否与当前实现一致（入参、出参、请求体、环境变量、排查步骤）；过时或重复的可标注「待归档」或合并。
- **系统规则**：核对是否仍为当前团队流程与约束；若流程变更，需同步改 `.cursor/rules` 或对应 docs。
- 在「人工检查 □」列打勾或注明「已查 / 待更新 / 归档」即可。
