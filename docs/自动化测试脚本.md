## 🔍 浏览器控制台测试命令

在浏览器控制台（F12）中执行以下命令进行快速测试：

### 检查登录状态

```javascript
// 检查token
localStorage.getItem('token')

// 检查用户信息
localStorage.getItem('userId')
localStorage.getItem('userRole')
localStorage.getItem('userName')
```

### 检查API连接

```javascript
// 测试后端连接
fetch('http://localhost:3000/health')
  .then(r => r.json())
  .then(console.log)
  .catch(console.error)
```

### 清除登录状态

```javascript
// 清除所有登录信息
localStorage.removeItem('token')
localStorage.removeItem('userId')
localStorage.removeItem('userRole')
localStorage.removeItem('userName')
location.reload()
```

### 检查布局偏好

```javascript
// 查看布局偏好
JSON.parse(localStorage.getItem('layoutPreferences') || '{}')
```

### 修改布局偏好（测试）

```javascript
// 设置列数为负数（测试范围保护）
localStorage.setItem('layoutPreferences', JSON.stringify({
  storeListCols: -5,
  statsCols: 20,
  taskListCols: 0
}))
location.reload()
```

## 🐛 常见问题排查

### 问题1: 服务无法启动

**检查**：
1. Node.js 是否安装：`node --version`
2. 端口是否被占用：`netstat -ano | findstr ":3000 :5173"`
3. 依赖是否安装：`cd backend && npm list`

**解决**：
```bash
# 清理端口
taskkill /F /PID [进程ID]

# 重新安装依赖
cd backend && npm install
cd ../frontend && npm install
```

### 问题2: 登录后跳转失败

**检查**：
1. 浏览器控制台是否有错误
2. localStorage 是否有 token
3. 后端服务是否在线

**解决**：
```javascript
// 清除所有缓存
localStorage.clear()
sessionStorage.clear()
location.reload()
```

### 问题3: 品牌策略生成失败

**检查**：
1. 后端服务是否在线
2. 是否选择了店铺
3. 是否填写了店铺属性

**解决**：
1. 检查后端日志
2. 检查网络请求（F12 -> Network）
3. 检查API响应

### 问题4: 布局设置不保存

**检查**：
1. localStorage 是否可用
2. 浏览器是否禁用localStorage
3. 是否有存储空间限制

**解决**：
```javascript
// 测试localStorage
try {
  localStorage.setItem('test', 'test')
  localStorage.removeItem('test')
  console.log('localStorage可用')
} catch (e) {
  console.error('localStorage不可用:', e)
}
```
