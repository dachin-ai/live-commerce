## 七、非弹窗但为浮层

| 说明 | 组件/位置 | 说明 |
|------|-----------|------|
| Toast 提示 | `ToastContext.tsx` | fixed top-4 right-4，z-[9999] |
| 错误边界页 | `ErrorBoundary.tsx` | 全屏居中卡片 max-w-md，非 overlay |
| 登录/注册/找回密码页 | Login / Register / ForgotPassword / ResetPassword | 整页表单，非弹窗 |

## 建议

- **宽度统一习惯**：简单确认/单表单用 `max-w-md` 或 `max-w-lg`；多表单项用 `max-w-2xl` 或 `max-w-4xl`。
- **z-index**：普通弹窗 50，侧栏触发的全局弹窗 100，Toast 9999。
- **交互**：所有模态框建议支持「点击遮罩关闭」和「Esc 关闭」（删除等危险操作可二次确认再关）。
