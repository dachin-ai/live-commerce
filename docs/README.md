# 项目文档索引

本目录存放直播电商中台管理系统的各类说明与测试文档。**已按「当前必读 / 常用 / 归档」分类整理**，归档文档位于 `归档/` 下子目录。

---

## 为什么根目录还有这么多文件？

根目录保留的约 **60+ 个文件**是**刻意设计**，不是漏归档：

- **当前必读、项目结构、测试与调试、LLM/待办/话术（常用）** 都放在根目录，方便：
  - 代码、`.cursor` 规则、Skill 里直接用 `docs/xxx.md` 引用，无需改路径；
  - 新人打开 `docs/` 就能按 **README 本页的分类表** 找文档，不必进子目录。
- **历史报告、过程记录、已停用协议** 已移入 `归档/`，根目录只保留**当前要用的**。

**查找方式**：不必在文件列表里翻——看本 README 的「当前必读」「项目与结构」「测试与调试」「LLM/待办/话术」表格，点击链接即可。归档内容见 [归档/README.md](./归档/README.md)。
