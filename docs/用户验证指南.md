## 🐛 常见问题

### Q1: 为什么没有生成任务？
**A**: 可能是以下原因：
1. 所有指标都达标且无异常（这是好事！）
2. 任务已存在（去重逻辑生效）
3. 没有统计数据（需要先上传数据）

**解决方案**:
- 检查后端日志（查看具体原因）
- 如果任务已存在，先点击"一键完成"清空旧任务

---

### Q2: 如何查看任务来源？
**A**: 在后端终端日志中查看：
```
[创建任务] 提升转化率至 3.8% (来源: threshold)
[创建任务] 完成首月10场直播 (来源: stage)
[创建任务] 【节日提醒】双11倒计时10天 (来源: event)
```

---

### Q3: 动态阈值是如何计算的？
**A**: 
```
转化率最低阈值 = max(历史平均 × 0.9, 行业基准 × 0.8)
转化率目标 = max(历史平均 × 1.1, 行业基准)

示例（greenpet）:
- 历史平均转化率: 1.93%
- 行业基准（宠物用品）: 3.8%
- 最低阈值: max(1.93 × 0.9, 3.8 × 0.8) = max(1.74, 3.04) = 3.04%
- 目标: max(1.93 × 1.1, 3.8) = max(2.12, 3.8) = 3.8%
```

