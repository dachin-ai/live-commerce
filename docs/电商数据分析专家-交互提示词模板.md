# 🤖 电商数据分析专家 - 交互提示词模板

## 👋 开场欢迎语

```
你好！我是你的电商数据分析专家助手，专注于帮助运营团队进行数据复盘和业务优化。

我可以帮助你：
📊 分析店铺数据（供应链、物流、定价、利润、渠道、营销、品牌）
🔍 搜索行业信息（平台规则、行业趋势、节庆活动、竞品洞察）
🎨 生成营销素材（产品图片、海报、图表、Logo）
📝 制作专业文档（Word报告、Excel表格、PPT演示）
🎬 优化直播间场景（分析现有场景、生成优化方案）
🎙️ 生成主播话术（完整话术、场景指南、话术优化）

请告诉我你需要什么帮助！
```

## 📦 常见使用场景模板

### 场景1：店铺数据分析

```
请帮我分析我的店铺数据，信息如下：
- 订单数：150单
- 用户数：2000人
- 销售额：22500元
- 店铺阶段：起步期

请给出：
1. 店铺阶段分析
2. 核心关注指标
3. 优化建议
4. 后续行动计划
```

### 场景2：多维度拆解分析

```
请帮我分析供应链维度，我的数据如下：
- 库存周转率：1.5
- 缺货率：8%
- 供应商数量：1家
- 供货周期：10天
- 库存成本占比：35%

请给出详细的分析和建议。
```

### 场景3：生成营销图片

```
请帮我生成以下营销素材：

1. 产品图片
   - 产品描述：白色简约款纯棉T恤，模特穿着展示
   - 风格：ins风

2. 营销海报
   - 主题：双十一促销
   - 产品类型：服装
   - 文字内容：全场5折起
```

### 场景4：生成专业文档

```
请帮我生成以下文档：

1. Word报告
   - 标题：店铺运营分析报告
   - 章节内容：
     * 第一章：数据概览 - 本月订单数100单，销售额5万元，同比增长20%
     * 第二章：优化建议 - 建议提升转化率，优化流量结构

2. Excel表格
   - 工作表名称：销售数据
   - 列：月份、订单数、销售额
   - 数据：
     * 1月，100单，10000元
     * 2月，120单，12000元
     * 3月，150单，15000元

3. PPT演示文稿
   - 标题：电商运营复盘
   - 内容：
     * 第一页：数据概览（本月订单数100单，销售额5万元，同比增长20%）
     * 第二页：优化建议（提升转化率、优化流量结构、增加用户留存）
```

### 场景5：直播间场景优化

```
请帮我优化直播间场景：
- 当前场景图片URL：https://example.com/live-scene.jpg
- 产品类别：美妆
- 主播风格：简约时尚

请分析现有场景，生成优化方案和效果图片。
```

### 场景6：生成主播话术

```
请帮我生成一套主播话术：
- 产品名称：温和氨基酸洁面乳
- 产品类别：美妆
- 产品特点：温和不刺激、深层清洁、保湿不紧绷
- 目标人群：敏感肌女性
- 价格范围：89-129元

### 场景7：搜索行业信息

```
请帮我搜索以下信息：

1. 淘宝中国平台最新规则和政策
2. 2025年中国电商节庆活动信息
3. 美妆行业在中国的最新趋势和热点

### 场景8：综合分析报告

```
请帮我生成一份综合的店铺运营分析报告，包含以下信息：

【基础数据】
- 订单数：150单
- 用户数：2000人
- 销售额：22500元

【渠道数据】
- 淘宝：营收10万元，流量10000
- 抖音：营收5万元，流量5000
- 其他：营收2万元，流量2000

【供应链数据】
- 库存周转率：1.5
- 缺货率：8%

【物流数据】
- 配送时长：6天

- 退货率：8%
- 物流满意度：3.5分

【定价数据】
- 平均客单价：150元
- 竞品平均价格：120元
- 转化率：3%
- 折扣率：20%

请给出：
1. 各维度详细分析
2. 问题识别和优先级
3. 优化建议和行动计划
4. 生成Word分析报告
5. 生成PPT演示文稿
```

## 💡 最佳实践提示

### 提示1：提供完整信息
```
❌ 不好的示例：
帮我分析一下我的店铺。

✅ 好的示例：
请帮我分析我的店铺数据：
- 订单数：150单
- 用户数：2000人
- 销售额：22500元
- 所属品类：美妆
- 当前阶段：起步期
```

### 提示2：明确分析维度
```
❌ 不好的示例：
帮我分析一下物流。

✅ 好的示例：
请帮我分析物流维度，我的数据如下：
- 配送时长：6天
- 准时送达率：85%
- 退货率：8%
- 物流满意度：3.5分
```

### 提示3：要求具体输出
```
❌ 不好的示例：
帮我生成个报告。

✅ 好的示例：
请帮我生成一份Word格式的店铺运营分析报告：
- 标题：店铺运营分析报告
- 包含章节：数据概览、问题分析、优化建议
- 保存到本地
```

### 提示4：结合多维度分析
```
❌ 不好的示例：
我的转化率低，怎么办？

✅ 好的示例：
我的店铺转化率只有3%，请帮我从以下维度分析：
1. 定价维度（平均客单价150元，竞品120元）
2. 内容营销维度（粉丝5000，互动率2%）
3. 物流维度（满意度3.5分，退货率8%）
给出综合优化建议。
```
