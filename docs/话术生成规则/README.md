# 话术生成规则文档索引

本目录下每种话术类型对应一份独立规则文档，便于查看该类型下**实际会发给 LLM 的提示词内容**与专属规则。  
逻辑实现见：`backend/src/rules/scriptResearch.ts`（市调与提示词拼接）、`backend/src/rules/scriptGeneration.ts`（类型元数据）。

| 话术类型 | 文档名 | 说明 |
|---------|--------|------|
| 人群互动话术 | [人群互动话术.md](./人群互动话术.md) | 暖场、活跃气氛、回答评论，提升互动率 |
| 场景化塑品话术 | [场景化塑品话术.md](./场景化塑品话术.md) | 产品介绍、建立使用场景、展示价值 |
| 促销活动话术 | [促销活动话术.md](./促销活动话术.md) | 折扣、比价、套餐，促进转化与客单价 |
| 逼单技巧话术 | [逼单技巧话术.md](./逼单技巧话术.md) | 限时限量、促单成交、制造紧迫感 |
| 完整销售流程话术 | [完整销售流程话术.md](./完整销售流程话术.md) | 从圈人群到逼单的完整转化漏斗 |

每份文档包含：
- **类型与目标**：该类型的目标、适用场景、建议时长
- **必须包含的要素**：话术结构中须覆盖的环节
- **该类型专属规则**：仅适用于该类型的约束与建议
- **系统提示词**：发给 LLM 的系统提示全文（含该类型配置）
- **用户消息模板**：任务行格式、市调摘要说明、结尾句
- **通用规则与禁止输出**：合规、语种、数字泛化及禁止出现的内容
