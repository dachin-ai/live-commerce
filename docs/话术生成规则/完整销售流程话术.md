# 完整销售流程话术 - 提示词与规则

对应类型 id：`full-sales`。覆盖从圈人群到逼单的完整转化漏斗，适合单品深度推销与高价值商品。

---

## 类型与目标

| 项目 | 内容 |
|------|------|
| 目标 | 完整转化漏斗，从圈人群到逼单 |
| 适用场景 | 单品深度推销、高价值商品 |
| 建议时长 | 90-180 秒 |

---

## 必须包含的要素

1. **圈人群/塑品**：明确「谁需要」「在什么场景需要」，并带出产品价值
2. **卖点提炼**：核心卖点 1～2 个重点讲，次要卖点补充，有主次
3. **打消顾虑**：常见顾虑（质量、售后、适用性）简短回应
4. **利益点**：价格、赠品、活动等利益说清楚（可结合用户填写的 promoCopy）
5. **售后保障**：退换、质保、客服等一句话带过
6. **逼单**：紧迫感 + 明确 CTA，收束整段话术

---

## 该类型专属规则

- **结构清晰、可分段照念**：建议按环节标注【圈人群】【塑品】【打消顾虑】【利益点】【售后】【逼单】，便于主播按段落使用。
- **承上启下**：各环节之间要有自然过渡句（如「好，说完为啥需要，接下来重点说好在哪里」「那大家最关心什么？我挑几个说」），避免生硬切段。
- **新人主播友好**：关键处可标注「此处可停顿看公屏」「这里可以举起来展示」，便于新人理解与执行。
- **场景化与代入感**：每个环节尽量用场景/痛点/故事/画面带出，禁止整段罗列卖点词；核心卖点与次要卖点分层呈现。

## 系统提示词（实际发送给 LLM）

```
你是直播话术策划。只输出「可念的话术正文」，禁止输出：市场分析、策略建议、优先级建议、行动清单、报告、中英对照列表等非话术内容。
当前生成「完整销售流程话术」。目标：完整转化漏斗，从圈人群到逼单。场景：单品深度推销、高价值商品。须包含要素：圈人群/塑品、卖点提炼、打消顾虑、利益点、售后保障、逼单。
通用规则:1)依摘要输入/店铺/合规/品类 2)禁词禁用慎词有据 3)语种与摘要一致 4)数字泛化 5)自然口语 6)紧扣产品与品类卖点 7)场景/痛点/故事带出卖点
```

（若有品类循环框架，会追加：`有[框架]时先出1～6模块框架(5～15分钟循环)，再润色为可念稿。`）

## 用户消息模板

**任务行：**

```
【任务】仅生成「完整销售流程话术」，产品：{产品名称}。目标：完整转化漏斗，从圈人群到逼单。请按要素（圈人群/塑品、卖点提炼、打消顾虑、利益点、售后保障、逼单）组织内容，只输出主播可念的话术正文，不要输出分析、建议、报告、清单等非话术内容。
```

**随后：** 市调摘要（含 [输入][店铺][合规][品类][要求]，以及若有 [框架]）。**结尾：** `请依上仅输出话术正文。` 或 `请先出框架再出可念稿。`

---

## 通用规则与禁止输出

- **通用**：合规、语种、数字泛化、自然口语、紧扣产品与品类、场景/痛点/故事带出卖点；完整流程话术还须「结构清晰、承上启下、可标环节与演示提示」。
- **禁止输出**：市场分析、策略建议、报告、行动清单、中英对照等非话术内容。
