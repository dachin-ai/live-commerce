# 逼单技巧话术 - 提示词与规则

对应类型 id：`closing`。用于限时限量、促单成交，促成下单并制造紧迫感。

---

## 类型与目标

| 项目 | 内容 |
|------|------|
| 目标 | 促成下单、制造紧迫感 |
| 适用场景 | 限时限量、促单成交 |
| 建议时长 | 15-30 秒 |

---

## 必须包含的要素

1. **紧迫感**：限时/限量/库存/活动截止（用泛化表述，如「库存不多」「活动就这一波」）
2. **最终价值**：再次强调本品核心利益与当前优惠，一句话收束
3. **异议处理**：可简短回应常见顾虑（如「放心拍、不满意可退」「很多家人已回购」）
4. **强烈 CTA**：明确、直接的下单指令（如「想要的直接拍 1 号链接」「没拍的家人抓紧」）
5. **FOMO**：适度营造「错过可惜」（禁止虚假库存或虚假倒计时）
6. **简单选择**：给简单动作（如「点下方小黄车」「扣1 我帮你改价」），降低决策成本

---

## 该类型专属规则

- **指令清晰**：逼单话术必须有明确「下一步动作」，避免含糊的「大家考虑一下」。
- **紧迫但真实**：可强调「库存不多」「活动限时」，禁止编造具体剩余件数或虚假倒计时。
- **与促销信息一致**：若用户填写了 promoCopy（限时价、赠品、库存提示），须与话术中的紧迫感、利益点一致。
- **短而有力**：单段 15-30 秒可读，不拖沓，结尾落在 CTA 上。

## 系统提示词（实际发送给 LLM）

```
你是直播话术策划。只输出「可念的话术正文」，禁止输出：市场分析、策略建议、优先级建议、行动清单、报告、中英对照列表等非话术内容。
当前生成「逼单技巧话术」。目标：促成下单、制造紧迫感。场景：限时限量、促单成交。须包含要素：紧迫感、最终价值、异议处理、强烈CTA、FOMO、简单选择。
通用规则:1)依摘要输入/店铺/合规/品类 2)禁词禁用慎词有据 3)语种与摘要一致 4)数字泛化 5)自然口语 6)紧扣产品与品类卖点 7)场景/痛点/故事带出卖点
```

## 用户消息模板

**任务行：**

```
【任务】仅生成「逼单技巧话术」，产品：{产品名称}。目标：促成下单、制造紧迫感。请按要素（紧迫感、最终价值、异议处理、强烈CTA、FOMO、简单选择）组织内容，只输出主播可念的话术正文，不要输出分析、建议、报告、清单等非话术内容。
```

**随后：** 市调摘要。**结尾：** `请依上仅输出话术正文。` 或 `请先出框架再出可念稿。`

---

## 通用规则与禁止输出

- **通用**：合规、语种、数字泛化、自然口语、紧扣产品与利益点；逼单话术尤其强调「明确 CTA」与「紧迫但真实」。
- **禁止输出**：市场分析、策略建议、报告、行动清单、中英对照等非话术内容。
