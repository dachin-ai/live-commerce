# 促销活动话术 - 提示词与规则

对应类型 id：`promotion`。用于折扣、比价、套餐推荐，促进转化与提升客单价。

---

## 类型与目标

| 项目 | 内容 |
|------|------|
| 目标 | 促进转化、提升客单价 |
| 适用场景 | 折扣、比价、套餐推荐 |
| 建议时长 | 20-40 秒 |

---

## 必须包含的要素

1. **成本与售价**：在合规前提下交代价格信息（可泛化，如「今天直播间价非常给力」）
2. **价格对比**：与日常价/竞品/线下对比，突出「划算」（需真实可验证，禁止「全网最低」等无依据说法）
3. **超值感**：赠品、加量、套装、满减等利益点
4. **紧迫感**：限时、限量、库存感（用泛化表述，如「库存不多」「活动就这一波」）
5. **省钱强调**：帮用户算账、强调省多少/多划算
6. **行动召唤**：明确的下单或加购引导

---

## 该类型专属规则

- **价格与活动需真实**：所有价格、折扣、赠品须与用户填写的「营销方案/促销活动」或市调摘要一致，禁止编造具体数字。
- **利益点前置**：用户若填写了 promoCopy（库存余量、倒计时、赠品、限时价等），须自然融入话术，不要遗漏关键利益点。
- **数字泛化**：避免「只剩 87 件」「已售 5000+」等硬编码，改用「库存不多」「已售出不少」「很多家人已下单」。
- **合规**：严禁「最好」「第一」「全网最低」等无依据用语；慎用词需有依据（如「很多用户反馈」）。

---

## 系统提示词（实际发送给 LLM）

```
你是直播话术策划。只输出「可念的话术正文」，禁止输出：市场分析、策略建议、优先级建议、行动清单、报告、中英对照列表等非话术内容。
当前生成「促销活动话术」。目标：促进转化、提升客单价。场景：折扣、比价、套餐推荐。须包含要素：成本与售价、价格对比、超值感、紧迫感、省钱强调、行动召唤。
通用规则:1)依摘要输入/店铺/合规/品类 2)禁词禁用慎词有据 3)语种与摘要一致 4)数字泛化 5)自然口语 6)紧扣产品与品类卖点 7)场景/痛点/故事带出卖点
```

---

## 用户消息模板

**任务行：**

```
【任务】仅生成「促销活动话术」，产品：{产品名称}。目标：促进转化、提升客单价。请按要素（成本与售价、价格对比、超值感、紧迫感、省钱强调、行动召唤）组织内容，只输出主播可念的话术正文，不要输出分析、建议、报告、清单等非话术内容。
```

**随后：** 市调摘要（含 [输入] 中的价、卖点、及若有 [营销/促销] 信息）。**结尾：** `请依上仅输出话术正文。` 或 `请先出框架再出可念稿。`

---

## 通用规则与禁止输出

- **通用**：合规、语种、数字泛化、自然口语、紧扣产品与品类；促销话术尤其要「价格与活动真实可验证」。
- **禁止输出**：市场分析、策略建议、报告、行动清单、中英对照等非话术内容。
