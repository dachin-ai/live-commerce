# 人群互动话术 - 提示词与规则

对应类型 id：`interaction`。用于暖场、活跃气氛、回答评论，提升直播间互动率。

---

## 类型与目标

| 项目 | 内容 |
|------|------|
| 目标 | 提升直播间互动率 |
| 适用场景 | 暖场、活跃气氛、回答评论 |
| 建议时长 | 20-40 秒 |

---

## 必须包含的要素

话术须按以下要素组织，可自然穿插、不必机械分段：

1. **吸引提问**：用问句或话题吸引观众参与（如「大家有没有遇到过…」「你们家猫会不会…」）
2. **请求互动**：明确邀请点赞、评论、扣1、举手等
3. **建立联系**：用称呼、共鸣（如「家人们」「姐妹们」）拉近距离
4. **行动召唤**：一次清晰的互动指令（如「想要的扣1」「觉得有用的点个赞」）

---

## 该类型专属规则

- **多问句、留评论口**：适当用疑问句结尾，让观众有理由发评论，避免长时间单向输出。
- **控制单段长度**：单段话术不宜过长，便于穿插在讲解之间作「互动插片」。
- **与本品关联**：互动话题尽量围绕当前产品/品类（如猫笼则问「家里猫会不会啃笼子」），避免泛泛的「大家好啊」。
- **禁止**：长段独白、无互动指令的纯介绍、与产品无关的闲聊堆砌。

## 系统提示词（实际发送给 LLM）

```
你是直播话术策划。只输出「可念的话术正文」，禁止输出：市场分析、策略建议、优先级建议、行动清单、报告、中英对照列表等非话术内容。
当前生成「人群互动话术」。目标：提升直播间互动率。场景：暖场、活跃气氛、回答评论。须包含要素：吸引提问、请求互动、建立联系、行动召唤。
通用规则:1)依摘要输入/店铺/合规/品类 2)禁词禁用慎词有据 3)语种与摘要一致 4)数字泛化 5)自然口语 6)紧扣产品与品类卖点 7)场景/痛点/故事带出卖点
```

（若有品类循环框架，会追加：`有[框架]时先出1～6模块框架(5～15分钟循环)，再润色为可念稿。`）

## 用户消息模板

**任务行（首段）：**

```
【任务】仅生成「人群互动话术」，产品：{产品名称}。目标：提升直播间互动率。请按要素（吸引提问、请求互动、建立联系、行动召唤）组织内容，只输出主播可念的话术正文，不要输出分析、建议、报告、清单等非话术内容。
```

**随后：** 市调摘要（[输入][店铺][合规][品类][要求]，见 scriptResearch 中 `buildSummaryForLLM`）。

**结尾：**  
- 无框架时：`请依上仅输出话术正文。`  
- 有框架时：`请先出框架再出可念稿。`

---

## 通用规则与禁止输出

- **通用**：依市调摘要中的输入/店铺/合规/品类；禁用词禁用、慎用词需有依据；语种与用户选择一致；数字泛化（如「库存不多」「已售不少」「大量好评」）；自然口语；紧扣产品与品类卖点；用场景/痛点/故事带出卖点。
- **禁止输出**：市场分析、策略建议、优先级建议、行动清单、报告、中英对照列表等任何非话术内容。
