# 完整销售流程话术说明

**类型标识**：`full-sales`  
**中文名称**：完整销售流程话术  
**适用场景**：单品深度推销、高价值商品、精细化运营  
**时长**：90-180 秒（可循环播放各环节）  
**规则文件**：`backend/src/rules/fullSalesScript.ts`

---

## 一、设计理念

### 1.1 设计来源

基于用户提供的**完整转化漏斗逻辑**，这是一套经过实战验证的单品推销流程，完整覆盖从"圈人群"到"逼单"的全链路。

### 1.2 核心特点

- **完整性**：不是单一环节话术，而是完整的销售闭环
- **可循环**：可拆分使用各环节，也可循环播放"塑品-打消顾虑-营销-售后-逼单"
- **互动性**：第三步"打消顾虑"需实时回复公屏，灵活调整
- **实战性**：融合心理学、价格锚定、从众效应、FOMO等转化机制

---

## 二、六步结构（完整转化漏斗）

### 第一步：圈人群 + 塑品（需求激发）

**时长**：20-30 秒  
**目标**：精准圈定目标人群，激发需求

**两类需求**：
1. **美好生活需求**：想要提升生活品质，获得更好体验（如"想给主子最好的生活"）
2. **现实不满需求**：对现状不满，想要改善问题（如"家里主子的用品不够好，总担心影响健康"）

**产品基础属性**：快速介绍产品定位（"今天给大家带来这款XX，专门为养宠家庭设计"）

**话术示例**（宠物产品）：
```
🎯 圈定目标人群（5-10秒）：
家人们，今天这款产品特别适合你们！

💭 需求激发（10-15秒）：
【美好生活需求】你们是不是也想给主子最好的生活，让它健康快乐每一天？
【现实不满需求】还是说你们现在家里主子的用品不够好，总担心影响它健康？

📦 产品基础属性（5秒）：
今天给大家带来这款猫宠，专门为养宠家庭设计的！
让我来告诉你们为什么这个是你们的必备！
```

---

### 第二步：卖点提炼（高效触达）

**时长**：25-35 秒  
**目标**：用分层卖点覆盖高、中、低意向用户

**核心卖点**（高意向用户）：
- 最核心的功能/优势，直击高意向用户痛点
- 建议做成**手卡或KT板**举起展示，视觉化呈现
- 可结合商品评论深挖（提示词："用客户容易理解的语言帮我提炼XX的核心卖点"）

**次要卖点1、2、3**（中低意向用户）：
- 性价比、设计、耐用性、便利性等
- 用"而且""还有"等词语自然衔接
- 尝试打动中低意向的犹豫用户

**话术示例**（宠物产品）：
```
⭐ 核心卖点（高意向用户，10-15秒）：
【核心卖点】不锈钢材质，安全无毒，主子用得放心！
这个是最重要的！高意向的家人们，你们最需要的就是这个功能！

📌 次要卖点（中低意向用户，15-20秒）：
【次要卖点1】耐用不易坏，一个能用好几年，省钱！
【次要卖点2】好清洗，不藏污纳垢，卫生有保障！
【次要卖点3】颜值高，拍照发朋友圈都好看！

💡 主播提示：
· 核心卖点建议做成手卡或KT板举起展示
· 可结合商品评论深挖卖点
· 用"你们看这个材质""闻这个味道"等实物展示增强说服力
```

---

### 第三步：打消顾虑（互动解答）

**时长**：20-30 秒  
**目标**：设置互动环节，引导出客户下单前顾虑并点对点解决

**常见顾虑类型**：
1. **SKU选择**："这么多规格/口味，我应该怎么选？"
2. **质量安全**："材质安全吗？会不会有异味/过敏？"
3. **效果担忧**："多久能见效？我用过很多产品都没用"
4. **对比竞品**："和XX大牌比怎么样？"

**互动话术**：
- "家人们，现在我来回答几个大家最关心的问题！"
- "公屏上有疑问的，赶紧打出来，我一个个给你们解答！"
- "不用担心选错，我来教你们怎么选最合适的！"

**话术示例**（宠物产品）：
```
❓ 互动环节（引导顾虑，20-30秒）：
家人们，现在我来回答几个大家最关心的问题！
公屏上有疑问的，赶紧打出来，我一个个给你们解答！

常见顾虑1：
问：这个材质安全吗？会不会有异味？
答：食品级不锈钢，无毒无味，通过质检，放心给主子用！

常见顾虑2：
问：我家猫/狗挑食，用新碗会不会不吃？
答：碗口设计符合宠物习惯，而且无异味，主子适应很快，评论里都说换了就用！

常见顾虑3：
问：有不同尺寸吗？我家是大型犬/小型猫。
答：有三个尺寸可选！小号适合猫和小型犬，中号适合中型犬，大号适合大型犬，看我打在公屏上的规格表！
```

---

### 第四步：利益点（价格优势）

**时长**：20-30 秒  
**目标**：展示营销活动的强力，制造超值感与紧迫感

**三大利益点**：

1. **赠品组套**：
   - 买一送三 / 买两件送礼盒
   - 强调赠品单独售价（"这个赠品单独卖都要XX元"）

2. **折扣力度**：
   - 原价 vs 直播间价格（自动计算锚定价）
   - 省钱金额（"省下的钱够你吃一顿大餐"）
   - 折扣比例（"相当于X折上车"）

3. **大促活动**：
   - 今天最后一天 / 今晚12点结束
   - 错过今天，下次大促要等XX
   - 全网最低价保证

**话术示例**：
```
💰 营销利益点（价格冲击，20-30秒）：

🎁 赠品组套：
今天下单就送超值大礼包！
买一送三：送价值68元的配套小件3件！
买两件再送价值98元的周边礼盒！

💸 折扣力度：
市场价748元，专柜要卖更贵！
但今天直播间，只要299元！只要299元！
直接省449元，相当于4折上车！

🔥 大促活动：
注意！这是我们今年最后一次大促！
今天晚上12点活动结束，明天来就没这个价了！

💡 主播强调：
"家人们，这个价格你们在外面绝对找不到！"
"我可以拍胸脯保证，这是全网最低价！"
```

---

### 第五步：售后保障（打消最后顾虑）

**时长**：10-15 秒  
**目标**：消除"货不对板"和"质量问题"的最后顾虑

**五大保障**：
1. **7天无理由退换**（运费我们承担）
2. **质量问题包换**（来回运费我们出）
3. **假一赔十**（保证正品）
4. **48小时发货**（顺丰/韵达包邮）
5. **专属客服**（售后小卡片二维码）

**售后小卡片内容**：
- 客服微信二维码（一对一服务）
- 使用说明/保养建议
- 晒单返现活动
- 复购优惠券

**话术示例**：
```
🛡️ 售后保障（打消最后顾虑，10-15秒）：

我知道很多家人担心：买回去万一不合适怎么办？

放心！我们有完善的售后保障：
✅ 7天无理由退换，运费我们承担！
✅ 质量问题包换，来回运费我们出！
✅ 假一赔十，保证正品！
✅ 48小时发货，全国包邮！
✅ 专属客服，扫售后小卡片二维码！

💡 主播强调：
"买得放心，用得安心，这是我们的承诺！"
```

---

### 第六步：逼单（重申+稀缺）

**时长**：15-20 秒  
**目标**：重新概括核心卖点+营销利益点+售后，强调库存稀缺性

**逼单四要素**：
1. **重申核心卖点**：1句话总结最核心功能
2. **重申营销利益点**：原价XX，今天XX，买一送三，今晚12点结束
3. **重申售后保障**：7天退换，假一赔十，48小时发货
4. **库存稀缺性**：只剩XX件，已卖出XX件，现在有XX人在看

**倒计时话术**：
```
我数三个数，数完之后我就下一个产品了！
3！准备好了吗？
2！点小黄车了吗？
1！下单！
```

**话术示例**：
```
🚀 逼单话术（重申+稀缺，15-20秒）：

好，家人们，我再给你们总结一遍：

📌 核心卖点：不锈钢材质，安全无毒，这是你们最需要的功能！
💰 营销利益点：原价748元，今天只要299元！买一送三！今晚12点活动结束！
🛡️ 售后保障：7天无理由退换，假一赔十，48小时发货！

⚠️ 库存稀缺性：
库存只剩87件了！刚才5分钟已经卖出去43件！
现在直播间有2000多人在看，你不抢，别人就抢走了！

⏰ 最后倒计时：
3！准备好了吗？
2！点小黄车了吗？
1！下单！手慢真的无！
```

## 三、话术循环（简化版）

完整流程讲完一遍后，可循环播放简化版：

```
【圈人群】→ 家人们，这款XX特别适合你们！

【塑品】→ XX材质，这是你们最需要的！

【打消顾虑】→ （回复公屏问题）质量有保证，7天退换！

【营销】→ 原价748元，今天299元，买一送三！

【售后】→ 48小时发货，假一赔十，放心买！

【逼单】→ 库存剩XX件，今晚12点恢复原价，赶紧下单！
```

**循环频率**：每个产品循环播放2-3次，然后进入下一个产品。

## 四、智能生成规则

### 4.1 产品类型自动识别

根据产品名称关键词，自动生成对应场景的话术：

| 产品类型 | 关键词 | 场景示例 |
|----------|--------|----------|
| 宠物产品 | 猫、宠、狗、pet | "家里主子的碗总是不干净，塑料的容易滋生细菌" |
| 美妆产品 | 护肤、美白、精华、面膜 | "早上起床照镜子，是不是发现脸色暗沉、毛孔粗大" |
| 食品产品 | 食品、零食、茶、咖啡 | "下午三点是不是又饿又困？办公室抽屉里的零食要么不健康" |
| 服装产品 | 衣、服装、鞋、包 | "衣柜满满但总觉得没衣服穿，好看的太贵，便宜的质量差" |
| 家居产品 | 家居、电器、厨具 | "家里东西不顺手，每次用都觉得不方便" |
| 数码产品 | 数码、手机、电脑 | "旧设备卡顿、功能不够，影响工作效率" |
| 通用产品 | 其他 | 自动生成适配的通用场景 |

### 4.2 价格自动计算

- **原价**：输入价格 × 2.8（锚定效应）
- **优惠金额**：原价 - 输入价格
- **折扣比例**：(输入价格 / 原价) × 10

示例：输入价格 299 元 → 原价 837 元，优惠 538 元，相当于 3.6 折

### 4.3 真实数据营造

- 已下单人数：随机 100-200 之间（"已有137位家人下单"）
- 库存剩余：随机 50-100 之间（"库存只剩87件"）
- 最近卖出：随机 30-50 之间（"刚才5分钟已经卖出去43件"）
- 评论数：5000+、4.9分等（营造从众效应）

## 五、使用建议

### 5.1 实战技巧

1. **核心卖点视觉化**：做成手卡或KT板，直播时举起展示
2. **结合商品评论**：用 DeepSeek 等 AI 分析评论，提炼真实用户关注点
3. **实时回复公屏**：第三步"打消顾虑"需灵活调整，针对性回答
4. **调整营销活动**：第四步根据实际活动修改赠品、折扣、大促
5. **拆分使用**：可单独使用某个环节（如只讲"卖点提炼"或"打消顾虑"）

### 5.2 适用场景

**推荐使用**：
- 高价值商品（客单价 > 100 元）
- 需要深度讲解的产品（功能复杂、卖点多）
- 精细化运营（有充足时间讲解单品）
- 直播间人气稳定（有足够观众停留）

**不推荐使用**：
- 快消低价品（如 9.9 元包邮）
- 时间紧张的直播（需快速过品）
- 产品卖点简单（无需深度讲解）
- 直播间人气不稳定（观众流失快）

### 5.3 时长控制

- **完整版**：90-180 秒（适合单品深度推销）
- **简化版循环**：30-40 秒（可循环播放2-3次）
- **单环节拆分**：每个环节 10-30 秒（可灵活组合）

## 七、总结

**完整销售流程话术**是基于实战转化漏斗设计的专业级话术，适合单品深度推销与精细化运营。它不是简单的"开场白+主要内容+结束语"模板，而是包含**圈人群、塑品、打消顾虑、利益点、售后、逼单**六个环节的完整闭环，每个环节都融入了心理学策略与实战技巧。

使用时建议结合商品评论分析、视觉化展示（手卡/KT板）、实时互动（回复公屏）等手段，最大化提升转化效果。
