# 抖店直播数据项目对标

本系统「数据筛选（Dashboard）」中的直播统计指标已对标**抖店电商罗盘 / 抖音直播数据**，在表格直接体现的 6 项核心指标之外，补充了**时效与派生类**数据项（**时效 = 每小时成交额**、人均观看时长、千次观看成交 GPM），与抖店罗盘常用维度对齐。

---

## 对标依据

- **抖店电商罗盘**：直播复盘、流量与转化、人均观看时长、千次观看成交金额（GPM）等。
- **行业共性**：核心成交与流量指标 + 派生效率指标（人均、千次） + **时效（每小时成交额）**。

---

## 一、核心指标（表格/看板直接展示）

与抖店直播复盘、TikTok Creator Center 导出列一致，本系统支持以下 6 项并在 Dashboard 中可勾选展示：

| 指标       | 说明           | 后端字段 / 计算           | 导入 Excel 列映射 |
|------------|----------------|---------------------------|-------------------|
| 成交额 GMV | 直播带货总成交 | `totalGMV`                | GMV、成交额等     |
| 直播时长   | 总时长（小时） | `totalDuration`           | 直播时长、时长(小时)等 |
| 观看人数   | 观看人次       | `totalViewers`            | 观看人数、观看人次等 |
| 互动数     | 评论/点赞等    | `totalInteractions`       | 互动数、互动等     |
| 订单数     | 总订单数       | `totalOrders`             | 订单数、订单等     |
| 转化率     | 转化率（%）    | `averageConversionRate`   | 转化率、转化率% 等；无列时按 订单/观看 推导 |

详见《TikTok数据导入使用指南》中的列别名与转化率计算逻辑。

## 二、时效与派生指标（对标抖店罗盘）

虽未在原始表格中单独成列，但抖店罗盘常用，本系统已实现并在看板中可选展示。

### 2.1 人均观看时长（分钟）

- **含义**：总直播时长（换算为分钟） / 观看人数，反映用户停留与内容吸引力。
- **后端**：`avgWatchDurationMinutes = (totalDuration * 60) / totalViewers`（无观看时为 0）。
- **前端**：数据项「人均观看时长」，单位分钟，支持环比变化展示。

### 2.2 时效（每小时成交额）

- **含义**：本系统中「时效」指**每小时成交额**（GMV ÷ 直播时长），衡量单位时间内的产出效率。
- **后端**：`gmvPerHour = totalDuration > 0 ? totalGMV / totalDuration : 0`。
- **前端**：在直播数据统计区域标题旁以标签展示「时效：¥xx,xxx/时」，悬停提示「每小时成交额（GMV ÷ 直播时长）」。

### 2.3 千次观看成交金额（GPM）

- **含义**：每千次观看产生的成交额，衡量流量变现效率。
- **后端**：`gpm = totalViewers > 0 ? (totalGMV / totalViewers) * 1000 : 0`。
- **前端**：数据项「千次观看成交 (GPM)」，货币格式化，支持环比变化展示。

## 四、扩展建议（可选）

- **更多派生指标**：如「观看-成交转化率」「客单价」等，可在后端聚合后增加字段与前端数据项。
- **实时/准实时**：若后续接实时数据源，可增加「实时」与「T+1 汇总」的切换或标识。
- **与抖店 API 对齐**：若对接抖店开放平台直播接口，可对照其字段名与口径，统一命名与计算方式。
