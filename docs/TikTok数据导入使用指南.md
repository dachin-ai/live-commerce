# TikTok数据导入使用指南

## 📋 功能说明

本功能支持从Excel文件导入TikTok平台的直播数据，自动计算统计数据，并基于数据生成智能待办事项。

**仪表盘 6 项数据抓取**：导入时会抓取并聚合 Excel 中与仪表盘「直播数据统计」对应的全部 6 项：总成交额(GMV)、总直播时长、总观看人数、总互动数、总订单数、平均转化率。若 Excel 含有「转化率」列，将参与平均转化率计算；若无则按「成交订单/观看人数」推算。

---

## 🚀 快速开始

### 1. 安装依赖

在项目根目录执行：

```bash
cd backend
npm install
```

这将安装 `xlsx` 库用于Excel文件解析。

### 2. 更新数据库结构

运行数据库迁移脚本以创建新表：

```bash
npm run db:update-seed
```

### 3. 启动服务

使用 `快速启动.bat` 启动前后端服务，或手动启动：

```bash
# 后端
cd backend
npm run dev

# 前端（新终端）
cd frontend
npm run dev
```

## 🎯 使用步骤

### 步骤1：准备Excel文件

1. 确保Excel文件包含TikTok直播数据
2. 第一行为表头（支持中英文）
3. 数据从第二行开始
4. 文件大小不超过50MB

### 步骤2：选择店铺

1. 登录系统
2. 在Dashboard顶部选择TikTok或抖音平台店铺
3. 如果店铺平台不是TikTok/抖音，"导入数据"按钮不会显示

### 步骤3：导入数据

1. 点击"导入数据"按钮（绿色按钮，位于"创建店铺"按钮旁边）
2. 在打开的模态框中点击"选择文件"
3. 选择Excel文件
4. 点击"开始导入"
5. 等待导入完成

### 步骤4：查看结果

导入成功后：
- 显示导入摘要（数据条数、总GMV、总观看人数、场次等）
- Dashboard统计数据自动更新
- 可在"数据分析"页面查看详细分析

### 步骤5：生成待办事项

1. 导入数据后，点击"智能生成刷新"按钮
2. 系统基于导入的数据自动生成量化待办事项
3. 待办事项包含：
   - 当前数值
   - 目标数值
   - 预计效果
   - 具体改进措施

## 📈 自动计算的统计数据

导入数据后，系统自动计算以下指标：

- **总成交额（GMV）**：所有直播的GMV总和
- **总直播时长**：所有直播时长总和（小时）
- **总观看人数**：所有直播观看人数总和
- **峰值观看人数**：峰值观看人数总和
- **总互动数**：所有互动数总和
- **总订单数**：所有订单数总和
- **成交订单数**：成交订单数总和
- **直播场次**：数据条数
- **平均每场时长**：总时长 / 场次
- **平均转化率**：成交订单数 / 总观看人数 × 100%
- **每小时GMV**：总GMV / 总时长
- **日均直播时长**：总时长 / 天数
- **日均场次**：场次 / 天数

## 🔍 待办事项生成逻辑

系统会根据导入的数据自动生成以下类型的待办事项：

### 1. 转化率优化
- **触发条件**：转化率 < 3% 且观看人数 > 100
- **内容**：包含当前转化率、目标转化率、预计增加订单数、预计GMV提升、具体措施

### 2. 直播时长优化
- **触发条件**：时长 < 20小时 或 GMV/小时 < 5000
- **内容**：包含当前时长、目标时长、预计GMV提升、建议时间段

### 3. 互动率优化
- **触发条件**：互动率 < 10% 且观看人数 > 50
- **内容**：包含当前互动率、目标互动率、预计增加互动数、具体措施

### 4. 客单价优化
- **触发条件**：客单价 < 200元 且订单数 > 10
- **内容**：包含当前客单价、目标客单价、预计GMV提升、具体措施

### 5. 观看人数增长
- **触发条件**：观看人数 < 500 且时长 > 10小时
- **内容**：包含当前观看人数、目标观看人数、预计增加人数、具体措施

## ⚠️ 注意事项

1. **文件格式**：只支持Excel文件（.xlsx, .xls）
2. **平台限制**：仅支持TikTok和抖音平台店铺
3. **数据验证**：系统会自动验证数据格式和完整性
4. **数据覆盖**：每次导入会创建新的统计数据记录，不会覆盖历史数据
5. **文件大小**：建议文件大小不超过50MB

## 🐛 常见问题

### Q1: 导入失败，提示"Excel文件中没有有效数据"
**A**: 检查Excel文件：
- 确保第一行是表头
- 确保从第二行开始有数据
- 确保至少有一个可识别的字段

### Q2: 导入失败，提示"该接口仅支持TikTok/抖音平台数据导入"
**A**: 确保选择的店铺平台是"TikTok"或"抖音"

### Q3: 统计数据计算不正确
**A**: 检查Excel文件中的数值格式：
- 确保数值字段是数字格式，不是文本
- 时长单位：列名为「直播时长」「时长(分钟)」等时按**分钟**（系统会转为小时）；列名为「时长(小时)」「直播时长(小时)」时按**小时**。详见《数据统计口径说明》（`docs/数据统计口径说明.md`）。
- 确保GMV和订单数等字段是有效数字

### Q4: 待办事项没有生成
**A**: 
- 确保数据导入成功
- 点击"智能生成刷新"按钮
- 检查是否有足够的统计数据
