# LLM 与智能生成 - 文档索引

本文档为大模型（LLM）、智能待办、话术生成相关说明的**入口索引**，便于快速找到当前配置、调用方式与历史归档。

---

## 一、当前入口与行为

| 项目 | 说明 |
|------|------|
| **智能待办（系统内）** | `POST /api/ai/generate-tasks`（body: `storeId`，可选 `useStatsFromStoreId`、`rawDailyTable`、`userPrompt`）。需登录。有数据且已配置 LLM 时优先走 Coze Agent / OpenAI，否则规则兜底。 |
| **智能体/第三方（API + Bot）** | `POST /api/ai/bot/generate-tasks`，Header `Authorization: Bearer {BOT_API_KEY}` 或 `X-API-Key`。Body：`storeId`（用该店铺 DB 数据）或 `rawDailyTable`+`metricsOverride`（Excel/按日 TSV + 汇总），可选 `userPrompt`。返回 `{ success, tasks }`，不写库。需配置环境变量 `BOT_API_KEY`。 |
| **话术生成** | 流式入口：`POST /api/ai/script/stream`。需登录且在「可使用 LLM 功能」名单内。按「LLM 调用方式」选 Coze Agent 或 OpenAI。 |
| **调用方式** | 仅两种：**Coze Agent**（stream_run）、**OpenAI 兼容接口**。管理员在「LLM 调用方式」页为待办/话术分别选择。 |
| **配置** | 话术 LLM 配置（API 地址、密钥）：管理员后台「话术 LLM 配置」或环境变量 `SCRIPT_LLM_URL`、`SCRIPT_LLM_API_KEY`。与待办共用，无单独 Agent 路由或密钥。 |
