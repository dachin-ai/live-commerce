# 产品分类对标抖店说明

本系统「创建店铺」中的**产品分类**（一级、二级、三级）已对标**抖店/抖音电商官方类目体系**，与抖店后台「店铺—资质中心—行业资质」及商品发布时的类目选择保持一致，便于商家按抖店类目经营、后续与抖店数据或入驻流程对齐。

## 对标依据

- **抖店官方**：抖音电商采用一级、二级、三级类目分层；本系统一级类目共 **18 个**，覆盖八大行业及宠物生活、母婴、医药保健、虚拟商品、本地生活、农资绿植、玩乐潮玩等常见一级类目。
- **数据来源**：类目名称与结构参考抖店官网类目选择说明、抖音电商学习中心及开放招商类目，二级/三级与抖店常见子类目一致。

## 二级类目示例（与抖店一致）

- **服饰鞋包**：女装、男装、内衣、鞋靴、箱包皮具、配饰  
- **美妆个护**：面部护肤、彩妆、香水、美发护发、个人护理  
- **食品健康**：休闲零食、生鲜、茶饮、酒类、保健食品  
- **亲子生活**：童装、玩具、孕产用品、喂养用品、教育培训  
- **家居家电**：家纺、家具、家电、家装建材、厨具  
- **3C数码**：手机、电脑、数码配件、智能设备、办公设备  
- 其余一级下二级类目见后端 `backend/src/db.ts` 分类种子数据。

## 三级类目

三级为更细粒度，与抖店商品发布时可选细分类一致（如女装下连衣裙、T恤、衬衫等），完整列表见 `backend/src/db.ts`。

## 使用说明

- **新建/重置库后生效**：类目种子在数据库**无任何分类数据**时才会插入。若你已有旧数据，需运行根目录 **「重置数据库.bat」** 后，新类目（如「美妆个护」「箱包皮具」）才会出现。
- **前端展示**：创建店铺弹窗中「产品分类」下拉与多选均从接口 `/api/categories` 拉取，无需改前端即可展示对标后的类目名称。
- **与抖店差异**：抖店类目会随平台运营调整，本系统按当前公开类目固化；若抖店后续新增/更名一级类目，可在 `db.ts` 中增改种子数据并重置数据库后同步。

## 维护与扩展

- 类目定义位置：`backend/src/db.ts`（`initDatabase` 内 `level1Categories`、`level2Categories`、`level3Categories`）。
- 新增一级：在 `level1Categories` 增加一行，再在 `level2Categories` / `level3Categories` 中为该一级补充二、三级。
- 更名：修改对应数组中的 `name`，重置数据库后生效。
