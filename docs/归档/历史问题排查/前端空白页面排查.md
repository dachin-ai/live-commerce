## 🔧 排查步骤

### 步骤 1: 检查浏览器控制台

**打开浏览器开发者工具：**
1. 按 `F12` 或 `Ctrl+Shift+I`
2. 切换到 "Console"（控制台）标签
3. 查看是否有红色错误信息

**常见错误类型：**
- `Cannot find module` - 模块导入错误
- `Cannot read property` - 属性访问错误
- `Unexpected token` - 语法错误
- `Network Error` - 网络请求失败

### 步骤 2: 检查网络请求

**在浏览器开发者工具中：**
1. 切换到 "Network"（网络）标签
2. 刷新页面（F5）
3. 查看是否有失败的请求（红色）
4. 特别关注：
   - `/api/auth/me` - 用户信息
   - `/api/stores` - 店铺列表

### 步骤 3: 检查后端服务

**确认后端服务运行：**
```powershell
# 检查后端是否在线
Invoke-WebRequest -Uri "http://localhost:3000/health"
```

**如果后端离线：**
- 运行 `单独启动后端.bat`
- 或运行 `快速启动.bat`

### 步骤 4: 检查登录状态

**检查是否已登录：**
1. 打开浏览器控制台
2. 输入：`localStorage.getItem('token')`
3. 如果返回 `null`，需要先登录

**登录方法：**
- 访问 http://localhost:5173/login
- 使用测试账号：
  - 管理员：admin@example.com / 123456
  - 普通用户：operator@example.com / 123456

### 步骤 5: 清除缓存并重新启动

**清除浏览器缓存：**
1. 按 `Ctrl+Shift+Delete`
2. 选择"缓存的图像和文件"
3. 清除数据

**清除 Vite 缓存：**
```powershell
cd "d:\Work space\lvbcsym\frontend"
Remove-Item -Recurse -Force node_modules\.vite -ErrorAction SilentlyContinue
```

**重新启动前端：**
```powershell
cd "d:\Work space\lvbcsym\frontend"
npm run dev
```

## 🐛 常见问题及解决方案

### 问题 1: 完全空白页面，控制台无错误

**可能原因：**
- React 组件没有正确渲染
- HTML 入口文件问题

**解决方案：**
1. 检查 `index.html` 中的 `<div id="root"></div>` 是否存在
2. 检查 `main.tsx` 是否正确导入
3. 查看浏览器 Network 标签，确认 `main.tsx` 是否加载成功

### 问题 2: 控制台显示 "Failed to fetch"

**可能原因：**
- 后端服务未启动
- API 请求失败

**解决方案：**
1. 确认后端服务运行在 `http://localhost:3000`
2. 检查 `vite.config.ts` 中的代理配置
3. 尝试直接访问 `http://localhost:3000/health`

### 问题 3: 控制台显示 "Cannot find module"

**可能原因：**
- 导入路径错误
- 文件不存在

**解决方案：**
1. 检查导入路径是否正确
2. 确认文件是否存在
3. 重新安装依赖：`npm install`

### 问题 4: 控制台显示 "useStore must be used within StoreProvider"

**可能原因：**
- `useStore` hook 在 `StoreProvider` 外部使用

**解决方案：**
- 确保所有使用 `useStore` 的组件都在 `StoreProvider` 内部
