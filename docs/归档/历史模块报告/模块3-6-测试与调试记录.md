## 🔧 环境准备

### 1. 依赖安装问题

**问题**：xlsx库安装失败
```
npm error code ENOTCACHED
npm error request to https://registry.npmmirror.com/@types%2fxlsx failed: cache mode is 'only-if-cached' but no cached response is available.
```

**原因**：npm配置为仅使用缓存模式，但缓存中没有xlsx库

**解决方案**：
1. 检查npm配置：`npm config get registry`
2. 如果需要，重置npm配置：`npm config delete registry` 或设置正确的镜像源
3. 重新安装：`npm install xlsx --legacy-peer-deps`

**临时方案**：代码中已添加错误处理，如果xlsx未安装会给出友好提示

### 2. 数据库更新

**状态**：需要运行 `npm run db:update-seed` 更新数据库结构

**注意**：如果遇到权限错误，可能需要：
- 关闭正在运行的数据库连接
- 以管理员身份运行
- 检查文件权限
