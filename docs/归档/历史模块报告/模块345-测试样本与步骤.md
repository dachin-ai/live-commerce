## 3. 模块 3：数据筛选（Dashboard）

- 在 Dashboard 点击 **「筛选数据」**，筛选面板应展开。
- 取消勾选「总直播时长」等某一项，对应统计卡片应隐藏。
- 切换时间周期（如「本周」），标签与数据周期一致。
- 点击「收起筛选」，面板应收起。

**通过标准**：筛选即时生效、展示与选项一致。

---

## 4. 模块 4：布局设置

- 点击右下角 **「布局设置」** 齿轮，布局设置面板打开。
- 关闭「店铺列表」等模块，对应区域应隐藏。
- 调整列数（如 2 列），布局应变为 2 列。
- 关闭「图标显示」，统计/趋势等图标应隐藏。
- 刷新页面（F5），设置应保持（localStorage）。
- 点击「重置为默认」，应恢复初始状态。

**通过标准**：布局即时生效、刷新不丢失、可重置。

## 5. 模块 5：智能任务生成

- 在 Dashboard 点击 **「智能生成刷新」**，应发起生成请求。
- 生成的任务标题应包含具体数值或目标（如「提升转化率至 x%」）。
- 任务描述应包含：当前值/目标值、至少 3 条具体措施。
- 紧急任务应排在前面并有标识。

**通过标准**：任务可生成、内容量化、措施可执行。

1. 先完成一次 **数据导入**（方式 A 或 B），确认 Dashboard 统计与 Excel 一致。
