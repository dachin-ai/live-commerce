**日期**：按执行日

## 一、步骤 1：按模块实现/确认后端功能代码

### 1.1 后端接口（已就绪）

| 接口 | 说明 | 状态 |
|------|------|------|
| GET /api/stats/live | 直播统计，支持 timeRange、storeId、month、year、dateFrom/dateTo | ✅ 已实现 |
| POST /api/data-import/tiktok | TikTok Excel 导入，按 storeId 写入 stats | ✅ 已实现 |
| GET /api/stores | 店铺列表（含 currency、currencySymbol） | ✅ 已实现 |
| GET /api/tasks | 任务列表，按 storeId 过滤 | ✅ 已实现 |

### 1.2 数据口径与单位

- 直播时长：支持 Excel「直播时长(秒)」「时长(小时)」等，导入后按小时汇总；仪表盘展示「本周期内合计」。
- 货币：按店铺货币存储；前端「显示为」支持店铺货币 + 人民币对多国（静态汇率）。

## 二、步骤 2：调用 MCP 测试前端，并对标市场成品方案

### 2.1 对标样本（模块 3）

- **淘宝/京东商家后台数据看板**：指标勾选、时间范围、实时生效。
- **行业共性**：筛选不刷新整页、选项与展示一致、可收起/展开。
- **抖音罗盘 / TikTok Creator**：直播指标、Excel 导入、看板更新、多国货币。

### 2.2 仪表盘功能清单（达模块2级别）

| 功能项 | 实现要点 | 自验 |
|--------|----------|------|
| 筛选面板 | 展开/收起「筛选数据」；数据项目、时间周期、收起筛选 | ✅ |
| 数据项目 | 9 项可勾选：GMV、直播时长、观看人数、互动数、订单数、转化率、时效、人均观看时长、GPM；至少保留一项（Toast 提示） | ✅ |
| 时间周期 | 今天/本周/本月/本季度/本年、选择月份、选择年份、自定义；周期标签含「本周期内合计」；选择月份/年份时显示具体年月；自定义时显示日期范围 | ✅ |
| 显示为 | 店铺货币 + 人民币对多国（CNY/USD/THB 等）；统计卡片与趋势图按所选货币换算展示 | ✅ |
| 统计卡片 | 按勾选展示；加载中/加载失败/暂无数据/空状态提示 | ✅ |
| 销售趋势图 | 按周期与店铺数据展示；Tooltip 按「显示为」货币格式化 | ✅ |
| 店铺隔离 | 切换店铺后统计、待办、趋势图均为当前店铺；顶部下拉与左侧列表同步 | ✅ |
| 导入数据 | 有 TikTok/抖音店铺时显示「导入数据」；弹窗展示当前店铺；选文件后可开始导入；导入成功后刷新统计与待办 | ✅ |
| 无店铺 | 提示「请选择店铺」与「创建新店铺」入口 | ✅ |
