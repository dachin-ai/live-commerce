## 🎯 快速测试指南

1. **启动服务**：运行 `快速启动.bat`
2. **访问应用**：http://localhost:5173
3. **测试品牌策略生成**：创建店铺 → 填写属性 → 点击"AI自动生成品牌策略"
4. **测试布局设置**：点击右下角设置按钮 → 调整布局
5. **测试数据筛选**：点击"筛选数据" → 选择数据项目和时间周期
