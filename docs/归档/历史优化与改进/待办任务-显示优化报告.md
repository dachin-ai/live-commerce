# 待办任务 - 显示优化报告

2026年2月2日

## 🎯 优化内容

### 用户反馈问题

1. ❌ **任务缺少店铺标识** - 无法区分是哪个店铺的待办事项
2. ❌ **"完善店铺信息"任务判断过于严格** - 即使已填写部分信息仍提示完善

---

## ✅ 解决方案

### 问题1：任务显示店铺名称

#### 后端修改（`backend/src/routes/tasks.ts`）

**原代码**:
```typescript
let query = 'SELECT * FROM tasks WHERE 1=1'
```

**修改后**:
```typescript
let query = 'SELECT t.*, s.name as storeName FROM tasks t LEFT JOIN stores s ON t.storeId = s.id WHERE 1=1'
```

**优势**:
- ✅ 通过LEFT JOIN关联stores表
- ✅ 自动获取店铺名称
- ✅ 对null storeId安全（返回storeName为null）

---

#### 前端修改（`frontend/src/components/TaskList.tsx`）

**原代码**:
```tsx
<p className="text-base font-semibold mb-2">
  {task.title}
</p>
```

**修改后**:
```tsx
<div className="flex items-center gap-2 mb-2">
  <p className="text-base font-semibold">
    {task.title}
  </p>
  {(task as any).storeName && (
    <span className="px-2 py-0.5 bg-blue-100 text-blue-700 text-xs font-medium rounded-full shrink-0">
      {(task as any).storeName}
    </span>
  )}
</div>
```

**效果**:
- ✅ 在任务标题旁边显示蓝色的店铺名称标签
- ✅ 只有绑定店铺的任务才显示标签（全局任务不显示）
- ✅ 标签不换行（shrink-0）

### 问题2：优化"完善店铺信息"判断逻辑

#### 后端修改（`backend/src/routes/ai.ts`）

**原逻辑**:
```typescript
// ❌ 只要有一个为空就提示
if (!storeInfo.targetAudience || !storeInfo.brandPositioning) {
  tasks.push({
    title: '完善店铺定位和目标人群',
    description: '请完善店铺的目标人群、品牌定位、价格区间等信息...',
    priority: 'normal',
  })
}
```

**问题分析**:
- greenpet店铺：
  - `targetAudience`: null（为空）
  - `brandPositioning`: "小型品牌"（有值）
  - `minPrice/maxPrice`: null（为空）
- 结果：即使已填写品牌定位，仍然生成任务 ❌

---

**新逻辑**:
```typescript
// ✅ 只有在缺失2个或以上关键信息时才提示
const missingInfoCount = [
  !storeInfo.targetAudience,
  !storeInfo.brandPositioning,
  !storeInfo.minPrice || !storeInfo.maxPrice,
].filter(Boolean).length

if (missingInfoCount >= 2) {
  const missingItems = []
  if (!storeInfo.targetAudience) missingItems.push('目标人群')
  if (!storeInfo.brandPositioning) missingItems.push('品牌定位')
  if (!storeInfo.minPrice || !storeInfo.maxPrice) missingItems.push('价格区间')

  tasks.push({
    title: '完善店铺定位和目标人群',
    description: `请完善店铺的${missingItems.join('、')}等信息，这将帮助系统提供更精准的运营建议和市场分析。完善后，系统将能够：1) 推荐最佳直播时段；2) 提供行业基准对比；3) 生成精准的市场分析。`,
    priority: 'normal',
    aiFeature: 'positioning',
  })
}
```

**优势**:
- ✅ 更宽松的判断条件（缺失2个或以上才提示）
- ✅ 动态描述缺失项目（如："请完善店铺的目标人群、价格区间等信息"）
- ✅ 更明确的价值说明（完善后能得到什么）

---

#### 判断逻辑对比

| 店铺状态 | 原逻辑 | 新逻辑 |
|---------|--------|--------|
| 目标人群✅ 品牌定位✅ 价格区间✅ | ❌ 不生成 | ✅ 不生成 |
| 目标人群❌ 品牌定位✅ 价格区间✅ | ❌ 生成（过于严格） | ✅ 不生成（只缺1个） |
| 目标人群❌ 品牌定位✅ 价格区间❌ | ❌ 生成 | ✅ 生成（缺2个） |
| 目标人群❌ 品牌定位❌ 价格区间✅ | ❌ 生成 | ✅ 生成（缺2个） |
| 目标人群❌ 品牌定位❌ 价格区间❌ | ❌ 生成 | ✅ 生成（缺3个） |
