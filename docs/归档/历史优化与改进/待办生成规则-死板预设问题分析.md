## 🎯 用户期望的流程（基于流程图）

```
用户上传数据
    ↓
表格异常数据抓取
    ↓
判断是否存在异常？
    ├─ Y（存在异常）→ 异常接入LLM分析生成建议和待办事项 → 抓取对应AI助手功能嵌入待办
    └─ N（不存在异常）→ 店铺属性的关联分析 → 基于属性和定位提供下一阶段目标的建议 
                         → 需要根据基本参数做市场调研和人群分析 → 生成待办事项
```

**核心理念**: 
1. **数据驱动**: 基于异常数据触发，而非固定阈值
2. **LLM分析**: 异常情况接入LLM生成个性化建议
3. **属性关联**: 正常情况基于店铺属性和定位生成阶段性目标
4. **市场调研**: 结合市场数据和人群分析

     const thresholds = {

       },

2. **趋势判断**:
   ```typescript
   function analyzeTrend(recentStats) {
     if (recentStats.length < 3) return 'insufficient_data'
     
     const trend = recentStats.map(s => s.gmv)
     const isDecreasing = trend.every((val, i) => i === 0 || val < trend[i-1])
     const isIncreasing = trend.every((val, i) => i === 0 || val > trend[i-1])
     
     if (isDecreasing) return 'declining'
     if (isIncreasing) return 'rising'
     return 'stable'
   }
   ```
