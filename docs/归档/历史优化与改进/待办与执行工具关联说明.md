# 待办事项与执行工具关联说明

本文说明**待办任务**与**执行工具**页中各工具的对应关系，便于运营从待办跳转或选用合适工具执行。以 Greenpet（测试-宠物）店铺的典型待办为例。

---

## 一、执行工具列表（当前系统）

| 工具 ID | 名称 | 说明 |
|--------|------|------|
| **speech** / **script** | 话术生成 | 输入产品与人群，基于店铺数据生成直播话术 |
| **report** | 生成报告 | 基于店铺数据生成周期复盘报告 |
| **market-analysis** | 市场分析 | 输入品类与时间范围，获取市场趋势与洞察 |
| **recommendations** | 商品推荐 | 基于店铺与品类生成商品推荐列表 |
| **stats** | 数据统计 | 拉取店铺指定周期数据并汇总展示 |
| **compare** | 店铺对比 | 选择至少 2 个店铺，对比 GMV、时长等指标 |
| **assistant** | AI助手 | 查看与处理当前店铺待办任务 |

后端待办中的 **aiFeature** 与上述工具的对应关系见下文「待办 → 执行工具」表。

---

## 二、Greenpet 典型待办 → 建议关联的执行工具

### 1. 【节日提醒】情人节倒计时7天-备货准备

**待办描述要点**：情人节 7 天后，礼品/美妆/鲜花品类增长，情侣礼盒、浪漫主题直播；建议配置话术 LLM 后点击「智能生成」获取个性化待办。

**建议关联执行工具**：

| 优先级 | 执行工具 | 用途 |
|--------|----------|------|
| 高 | **话术生成** (speech) | 生成情人节/浪漫主题直播话术、情侣礼盒讲解话术 |
| 高 | **商品推荐** (recommendations) | 备货选品：礼品、美妆、鲜花、情侣礼盒组合 |
| 中 | **数据统计** (stats) | 查看近期 GMV/订单，辅助备货量与排期 |
| 中 | **生成报告** (report) | 复盘上一大促或近期直播，指导本次节日策划 |

**后端 aiFeature**：节日类待办已统一设置 `aiFeature: 'event'`，前端可按此优先展示「话术生成」「商品推荐」。

---

### 2. 平抑GMV异常波动

**待办描述要点**：GMV 波动率达 200%，需针对低 GMV 场次（如 2025-12-25 仅 2701 泰铢）复盘内容与选品，建立场次目标管控机制，确保每日 GMV 偏差不超过 30%。

**建议关联执行工具**：

| 优先级 | 执行工具 | 用途 |
|--------|----------|------|
| 高 | **数据统计** (stats) | 按日/按场查看 GMV、观看、转化，定位低 GMV 场次 |
| 高 | **生成报告** (report) | 周期复盘，分析内容与选品对 GMV 的影响 |
| 高 | **商品推荐** (recommendations) | 优化选品，对应后端 aiFeature：product_recommend |
| 中 | **话术生成** (speech) | 优化话术与转化路径，对应后端 aiFeature：script |
| 中 | **市场分析** (market-analysis) | 看品类与市场趋势，辅助目标与偏差设定 |

**后端 aiFeature**：异常类待办已按类型设置（如 GMV 突降 → `product_recommend`，转化率 → `script`），前端可按 aiFeature 推荐上述工具。

## 三、待办来源与 aiFeature 对应表（后端）

| 待办来源/类型 | 典型 aiFeature | 建议首推执行工具 |
|---------------|----------------|------------------|
| 节日提醒 (event) | event | 话术生成、商品推荐 |
| GMV 异常 (anomaly) | product_recommend | 商品推荐、数据统计、生成报告 |
| 转化率异常/阈值 | script | 话术生成 |
| 观看人数异常/阈值 | time_recommend | 数据统计 |
| 互动率异常/阈值 | engagement | 话术生成（互动话术） |
| 客单价异常/阈值 | pricing | 商品推荐、市场分析 |
| 直播时长阈值 | schedule | 数据统计、生成报告 |
| 观看人数增长 | marketing | 市场分析、话术生成 |
| 冷启动/短视频引流 | content | 话术生成、商品推荐 |
| 完善店铺信息 | positioning | 店铺管理（非执行工具页） |
| 成长期周报/复购 | report, crm | 生成报告 |
| 成熟期品牌/供应链 | brand, supply_chain | 市场分析、生成报告 |

## 四、前端可做的增强（可选）

1. **待办卡片展示「推荐执行工具」**：根据 `task.aiFeature` 或标题关键词（如「节日」「GMV」「转化率」）显示 1～3 个执行工具入口（如「用话术生成」「用商品推荐」），跳转到执行工具页并可选带入店铺/品类。
2. **节日类 aiFeature**：后端已在生成「节日提醒」待办时写入 `aiFeature: 'event'`，前端可按此统一推荐话术生成、商品推荐。

---

## 五、小结

- **【节日提醒】情人节倒计时7天-备货准备**：主要关联 **话术生成**、**商品推荐**，其次 **数据统计**、**生成报告**。
- **平抑GMV异常波动**：主要关联 **数据统计**、**生成报告**、**商品推荐**，其次 **话术生成**、**市场分析**。

系统内待办若带有 **aiFeature**，前端即可按上表映射到执行工具并做「推荐工具」展示或跳转。
