## 🔄 数据库操作优化

### 之前的方式（3-4步）
1. 修改 `backend/src/db.ts` 中的种子数据
2. 删除 `backend/data.db`
3. 重启后端服务
4. 重新创建数据库

### 现在的方式（1步）
1. 修改 `backend/src/db.ts` 中的种子数据
2. 执行：`cd backend && npm run db:update-seed`
3. 完成（保留业务数据）

**优势**：
- 操作步骤减少 67%
- 保留业务数据
- 无需重启后端
