# 平台适配和任务维度 - 优化报告

## 📅 优化时间
2026年2月2日

## 🎯 用户反馈问题

### 问题1：平台数据源不匹配 ❌
**现象**: greenpet店铺选择的是TikTok，但转化率描述中显示"抖音/快手平台2024-2025年Q4大盘数据"

**原因**: 数据源描述硬编码，未根据店铺平台动态调整

---

### 问题2：缺少单场和主播维度任务 ❌
**现象**: 当前生成的待办事项主要是店铺整体维度的（如：提升转化率、增加直播时长），缺少针对单场直播和主播的任务

**用户需求**: 
- 单场直播复盘
- 主播话术优化
- 主播状态管理
- 主播培训建议

---

## ✅ 解决方案

### 修复1：平台自适应数据源描述

#### 修改函数签名
```typescript
// 原函数
function getConversionRateBenchmark(
  categories: string[], 
  minPrice?: number, 
  maxPrice?: number
): { rate: number; comparison: string }

// 新函数（添加platform参数）
function getConversionRateBenchmark(
  categories: string[], 
  minPrice?: number, 
  maxPrice?: number, 
  platform?: string
): { rate: number; comparison: string }
```

#### 平台映射逻辑
```typescript
// 根据平台生成对比描述
let dataSource = '抖音/快手平台2024-2025年Q4大盘数据' // 默认

if (platform === 'TikTok') {
  dataSource = 'TikTok平台2024-2025年Q4全球数据'
} else if (platform === '淘宝' || platform === '天猫') {
  dataSource = '淘宝/天猫平台2024-2025年Q4大盘数据'
} else if (platform === '抖音') {
  dataSource = '抖音平台2024-2025年Q4大盘数据'
} else if (platform === '快手') {
  dataSource = '快手平台2024-2025年Q4大盘数据'
}

const comparison = `低于「${categoryName}」品类基准 ${finalBenchmark}%${priceNote}（数据来源：${dataSource}）`
```

#### 调用点修改
1. **getDynamicThresholds函数** - 添加platform参数传递
2. **转化率优化任务生成** - 传入`storeInfo?.platform`

---

### 新增2：单场维度任务生成

#### 新增函数：`generateSingleSessionTasks()`

**生成的任务类型**:

1. **复盘最近一场直播**（总是生成）
```typescript
{
  title: '复盘最近一场直播',
  description: `【单场数据】GMV ¥XXX | 观众 XXX人 | 订单 XXX单 | 时长 XXh

📊 **关键指标**：
- 转化率：X.XX%
- 客单价：¥XXX
- 互动率：X.XX%

📝 **复盘要点**：
1) 本场直播的优势环节（哪些商品/话术效果最好）
2) 本场直播的问题环节（流量流失点在哪里）
3) 观众高频提问（反映需求和痛点）
4) 下场改进计划（3个具体措施）

💡 建议主播观看回放，记录优化点。`,
  priority: 'normal',
  aiFeature: 'review',
  source: 'session'
}
```

2. **改进本场转化率低的问题**（条件触发）
- **触发条件**: 本场转化率 < 历史平均 × 0.8
- **优先级**: urgent
- **包含内容**:
  - 可能原因分析（5个方向）
  - 改进措施（4个具体措施）

---

### 新增3：主播维度任务生成

#### 新增函数：`generateAnchorTasks()`

**生成的任务类型**:

1. **优化主播话术脚本**（总是生成）
```typescript
{
  title: '优化主播话术脚本',
  description: `【主播话术优化】

📝 **当前问题**：
- 话术可能缺乏吸引力（观看人数XXX人，互动率X.XX%）
- 缺少固定的开场白和结束语
- 商品介绍缺乏卖点提炼

✅ **优化方向**：
1) **开场白**：3句话吸引注意力（问候+福利预告+今日主题）
2) **商品话术**：FABE法则（特征-优势-利益-证据）
3) **互动话术**：每5分钟设置一次互动问题
4) **促单话术**：限时优惠、稀缺性、从众心理
5) **结束语**：感谢+预告下场+引导关注

💡 可使用AI助手生成专业话术模板。`,
  priority: 'normal',
  aiFeature: 'script',
  source: 'anchor'
}
```

2. **主播长时间直播状态管理**（条件触发）
- **触发条件**: 直播时长 > 4小时
- **包含内容**:
  - 注意事项（4个方面）
  - 建议方案（4个措施）

3. **主播互动技巧培训**（条件触发）
- **触发条件**: 互动率 < 10%
- **包含内容**:
  - 培训内容（5个技巧）
  - 培训方式（3种方法）
