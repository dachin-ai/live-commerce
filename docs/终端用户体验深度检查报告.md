## 二、本次修改项汇总

1. **Sidebar（收起态）**  
   - 原：`Link to="/"` + 文案「版本更新日志」。  
   - 现：`Link to="/messages"`，文案「站内信」，与展开态一致；增加未读角标与当前页高亮。

2. **AnalysisPage**  
   - 原：仅处理 `isLoading` 与 `stats`，统计接口失败时无提示。  
   - 现：使用 `isError: statsError`，统计失败时展示「统计数据加载失败，请检查网络或稍后重试」。
